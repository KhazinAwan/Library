# Library

A simple Library application built as part of The Odin Project's JavaScript curriculum.

The project allows users to add books to a personal library, view them as cards, remove books, and toggle their read status. The application demonstrates object constructors, prototype methods, DOM manipulation, event handling, and form processing using vanilla JavaScript.

---

## Features

- Add new books through a form
- Store books in a JavaScript array
- Automatically generate unique IDs for each book
- Display books as individual cards
- Remove books from the library
- Toggle a book's read status
- Client-side form validation using HTML attributes
- Responsive card layout using CSS Grid
